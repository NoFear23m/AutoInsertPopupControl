Führen Sie die folgenden Schritte aus, um das benutzerdefinierte Steuerelement in einer XAML-Datei in einem anderen Projekt zu verwenden:

  • Fügt dem Projekt einen Verweis hinzu.
  • Fügen Sie die folgende Zeile dem Stammelement der XAML-Datei hinzu, in der Sie das folgende Steuerelement verwenden möchten:
    xmlns:MyNamespace="clr-namespace:AutoInsertPopup;assembly=AutoInsertPopup"
  • Verwenden Sie das Steuerelement in der XAML-Datei:
    <MyNamespace:CustomControl1/>
